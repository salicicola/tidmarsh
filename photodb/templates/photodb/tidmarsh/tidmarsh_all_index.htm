Notes. Limitations of this draft version: nearly all images (excluding animals) retrieved from another server; all or nearly all pages are not mobile friendly. Number of taxa at the gallery differ from that of the checklist (623 taxa as of March 2018), because gallery includes only photos published at salicicola (573 taxa as of February 2021).
Tech info: container's size ~ 3.3MB (virtual 1.15GB). Requirements according to dockerfile, etc.: Django 3.0.8 and python 3.6 with Pillow, lxml, python-dateutil. (Dev version worked fine with slightly different versions, e.g. Python 3.6.9 & Django 3.1.7). May use persistent volume :/data which contains all images, SQLite DB and XML/TXT files for uncommitted records; however, keep in mind that updating container's version do not update existed files in the volume automatically. — Currently running Python {{python_version}} & Django {{django_version}}.
